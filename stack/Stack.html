Stack

A stack is a data structure that follows the Last In First Out (LIFO) principle.

Basic Operations

  1. : Add an element to the top of the stack.
  2. : Remove the top element from the stack.
  3. : Get the value of the top element without removing it.
  4. : Remove all elements from the stack.
  5. : Get the number of elements in the stack.
